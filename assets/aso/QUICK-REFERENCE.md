## Key Character Limits

### Apple App Store (iOS/macOS)

| Field            | Character Limit                  |
| ---------------- | -------------------------------- |
| App Name         | 30 characters                    |
| Subtitle         | 30 characters                    |
| Promotional Text | 170 characters                   |
| Description      | 4,000 characters                 |
| Keywords         | 100 characters (comma-separated) |
| What's New       | 4,000 characters                 |

### Microsoft Store

| Field             | Character Limit   |
| ----------------- | ----------------- |
| Short Description | 200 characters    |
| Description       | 10,000 characters |

### Google Play Store

| Field             | Character Limit  |
| ----------------- | ---------------- |
| Short Description | 80 characters    |
| Full Description  | 4,000 characters |

## Localization Notes

### Indonesian Market

- Use "Skandinavia" and "Nordic" interchangeably
- Keep "hygge" in Latin script (no Indonesian equivalent)
- Use 24-hour time format (15:00 not 3:00 PM)
- Emphasize family and group experiences
- Formal but warm tone

### English Market

- International English (not US-specific)
- Focus on unique experience and design
- Emphasize authenticity and quality
- Professional yet approachable tone

**Apple App Store Emoji Policy**: All Apple App Store versions (iOS and macOS) have emojis removed as they are not permitted in App Store Connect. Microsoft Store and Google Play versions retain emojis for better visual appeal and engagement.
